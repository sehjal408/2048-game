# 2048

A simple browser implementation of the classic 2048 puzzle game, built with vanilla HTML, CSS, and JavaScript.

## How to play

- Open `index.html` in your browser.
- Use the **arrow keys** (or **swipe** on touch devices) to slide all tiles in one direction.
- Tiles with the same number merge into one when they touch, doubling their value.
- Reach the **2048** tile to win — or keep going for a higher score!
- Click **New Game** to start over at any time.

## Running locally

No build step or dependencies required. Just open `index.html` directly in a web browser.

## Files

- `index.html` — page structure and styling
- `game.js` — game logic, rendering, and input handling
